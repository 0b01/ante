◆ getTagVal()

unsigned short ante::AnDataType::getTagVal ( std::string &  name)

Gets the name of the parent union type.

Will fail if this DataType is not a union tag and contains no fields. Use isUnionTag before calling this function if unsure.

Returns
The name of the DataType containing this UnionTag Returns the UnionTag value of a tag within the union type.

This function assumes the tag is within the type. The 0 returned on failure is indistinguishable from a tag of value 0 and will be changed to an exception at a later date.
