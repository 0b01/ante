class Triple {
    int first, second, third;

public:
    Triple(int a, int b, int c) :
        first(a), second(b), third(c){}

    int firstPlus(int d){
        return first + d;
    }

//in driver.an
import Cpp "file.hpp"
let t = Triple(1, 2, 3)
t.firstPlus t.third  //=> 4

Factorial

  fun fact: i32 x -> i32
    if x == 1 then 1
    else x * fact (x-1)

fact 5  //=> 120

Lazy Types

  fun when: Comparable cond, lazy 't a b c
    match cond with
    | LessThan -> a
    | Equal -> b
    | GreaterThan -> c

let rand = random 1 100
let input = i32.parse (input "Enter your guess: ") ?
    None -> error "Please enter an integer!"

when (input.compare rand) "too low!" "just right!" "too high!"

Strongly, Statically Typed

Ante’s algebraic type system can prevent entire classes of bugs so you can catch them before ever running your program. For example, null pointers are replaced with the much safer Maybe type, which prevents null pointer dereferences when used.

Extensible

A compile-time API for the compiler itself enables the creation of compiler extensions within the program itself. This enables anything from creating new optimization passes to extending the type system with dependent types.

Robust Module System

Modules provide an easy way to manage multiple files and reason about large systems. They also enable faster compilation through caching.

Integrated Build System

Build instructions can be specified within the source code itself, no makefile needed. Additionally, no special syntax is required as it is implemented as a normal library module using Ante’s compile-time API.

High & Low Level language

Ante hosts a variety of high-level solutions such as smart pointers and first-class functions but also provides the ability to interact at a lower level if needed.

Gradual Memory Management

To support the largest variety of applications, Ante provides several different methods of memory management, including manual management. Don’t want to bother with any of it? Simply import GC to use a garbage collector.
