Detailed Description

The result of a type check.

Can be one of three states: Failure, Success, or SuccessWithTypeVars.

SuccessWithTypeVars indicates the typecheck is only a success if a typevar within is bound to a particular type. For example the check of 't* and i32* would return this status. Whenever SuccessWithTypeVars is set, the bindings field contains the specific bindings that should be bound to the typevar term.
